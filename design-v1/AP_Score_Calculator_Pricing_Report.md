# Pricing Report: AP Score Calculator

## 1. 结论

**推荐方案：Free + Pro 订阅制（$4.99/月 或 $39/年）**

核心逻辑：
- 纯前端工具，**零边际成本**，每次计算不消耗 API
- 竞品（Fiveable、Albert.io、Marco Learning）均以**综合备考平台**形式收费，定价 $10-25/月，但分数计算器本身 universally **免费引流工具**
- 你的差异化是"最准确、最易用"，可以走**轻量 Pro 增值**路线
- 学生付费意愿中等，低价高频是最佳策略

最大风险：AP 考试有强季节性（3-5月旺季），全年订阅可能流失率高，需设计学期制或按需付费选项。

## 5. 收费模型选择

| 模型 | 适合度 | 理由 |
|---|---|---|
| **订阅制** | ⭐⭐⭐ 推荐 | 零边际成本，订阅是利润最大化的方式 |

## 6. 免费层设计

免费层是**转化漏斗入口**，不是慈善：

| 维度 | 设计 |
|---|---|
| 每日免费次数 | **无限次计算**（纯前端，无成本限制） |
| 功能限制 | 仅基础估分；Pro 解锁：历史记录、趋势分析、多科目对比、家长报告 |
| 注册要求 | 否，匿名可用基础版（降低摩擦） |
| 滥用防护 | IP 限流（每分钟 10 次），防脚本刷 |
| 广告 | 免费层底部展示非侵入式广告（可选） |

**免费层月成本：$0**（纯前端，无服务器计算）

## 8. 年付 / Lifetime 建议

**年付折扣：35%**（$4.99×12=$59.88 → $39/年）

理由：
- 零边际成本，折扣可以激进
- 学生群体年付意愿低，需要强刺激
- 定价心理学：$39 看起来比 $4.99×12 便宜很多

**Lifetime：✅ 可以做**

```
Lifetime 价格 = $4.99 × 18~24 = $89.99 ~ $119.99
```

推荐：**$99 Lifetime**

必须附加：
1. **Fair Use 条款**：限个人使用，禁止共享账号
2. **功能锁定**：Lifetime 仅包含购买时的功能，新功能可能需额外付费
3. **限量发售**：前 200 名，制造稀缺感
4. **退款政策**：7 天无理由（Stripe 支持）

## 9. Stripe 支付配置建议

| 字段 | 示例值 | 说明 |
|---|---|---|
| `product_key` | `ap_score_pro` | 产品标识 |
| `price_id_monthly` | `price_xxx` | Stripe Price ID（月付） |
| `price_id_annual` | `price_yyy` | Stripe Price ID（年付） |
| `price_id_semester` | `price_zzz` | Stripe Price ID（学期） |
| `price_id_lifetime` | `price_www` | Stripe Price ID（一次性） |
| `plan` | `pro` | 计划名称 |
| `billing_cycle` | `monthly` / `yearly` / `semester` / `lifetime` | 计费周期 |
| `features` | `["all_subjects", "history", "trends", "parent_report", "no_ads"]` | 功能列表 |
| `overage_allowed` | `false` | 无超额（纯前端无用量概念） |
| `entitlement_key` | `pro_access` | 权限开关 |

**Webhook 必须处理：**
- `checkout.session.completed` → 开通 Pro
- `customer.subscription.created` → 记录订阅
- `customer.subscription.updated` → 更新计划
- `customer.subscription.deleted` → 降级为 Free
- `invoice.paid` / `invoice.payment_failed` → 续费/催付

**Stripe 配置代码建议：**
```javascript
// 创建产品
const product = await stripe.products.create({
  name: 'AP Score Calculator Pro',
  description: 'Unlimited access to all AP subjects with history & trends',
});

// 月付
const monthly = await stripe.prices.create({
  product: product.id,
  unit_amount: 499, // $4.99
  currency: 'usd',
  recurring: { interval: 'month' },
});

// 年付
const yearly = await stripe.prices.create({
  product: product.id,
  unit_amount: 3900, // $39.00
  currency: 'usd',
  recurring: { interval: 'year' },
});

// Lifetime（一次性）
const lifetime = await stripe.prices.create({
  product: product.id,
  unit_amount: 9900, // $99.00
  currency: 'usd',
  // 无 recurring
});
```

## 11. 上线后验证指标

| 指标 | 健康范围 | 说明 |
|---|---|---|
| 免费→付费转化率 | **2-5%** | 工具站标准 |
| 月付 vs 年付比例 | 年付 20-40% | 学生偏好月付，家长偏好年付 |
| 月流失率 | **<8%** | 学生工具正常偏高 |
| 季节性波动 | 3-5月流量 3-5 倍 | 备考季自然高峰 |
| ARPU | 接近 $4.99 | 若年付多，ARPU 会更高 |
| NPS | >30 | 学生工具口碑传播关键 |

**调价信号：**

| 信号 | 动作 |
|---|---|
| 转化率 >8% | 价格偏低，试涨到 $6.99 |
| 转化率 <1% | 免费层太慷慨，或价格太高 |
| 流失率 >15% | 产品价值问题，非价格问题 |
| 学期制购买占比 >50% | 季节性策略有效，加大推广 |

| 风险 | 等级 | 应对 |

| 竞品计算器全免费，用户不愿付费 | 🔴 高 | Pro 功能必须明显优于免费层（历史趋势、家长报告） |

| 流量获取成本高 | 🟡 中 | SEO 核心关键词排名，依赖灵引 |

**下一步建议：**
1. **MVP 阶段**：免费层全功能上线，先拿流量和排名
2. **验证阶段**：观察 3-5 月自然流量和留存，再开 Pro
3. **付费层功能**：历史记录、趋势分析、家长邮件报告、多科目对比
4. **支付接入**：Stripe Checkout，先上月付+年付，学期制和 Lifetime 后续再加
